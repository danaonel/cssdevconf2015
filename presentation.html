Take-aways

Everything Sara Soueidan talked about SVG

Flexbox

  • it's becoming widely supported and it's the future
  • justify-content vs align-items
  • excellent demonstrations (Charlie Park is an excellent speaker)

Pattern Libraries

  • Think design systems, not pages
  • The culture around pattern libraries needs to change (educate designers, increase collaboration between devs and designers, have the entire team get involved)
  • Levels of maturity of pattern libraries (5 levels from static styleguide to automated pattern library)
  • Pattern library tools - npm install patternpack

Visual Regression testing

  • Various regression testing tools demos
  • phanton.js, webdriver.io, wraith.io, casper.js

SMACSS

  • Excellent breakdown of reasons on why we should write modular CSS
  • CSS should be scalable, encapsulated, predictable
  • The CSS should be organized in tiny bootstraps for each component
  • Conclusions: categorize CSS rules, use meaningful names, use shallow selectors
